Send parse push notifications from wordpress backend for android and ios.

== Description ==

Parse Push Wordpress is simple send parse push notifications from wordpress backend for android and ios.

* Custom Audience for your push notifications by:
	-Channels
	-Type of Device

* Send simple text message

= Website =
http://quiroa.me/parse-push-for-wordpress/

== Installation ==
1. Upload 'parse-push-wordpress' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "Parse Push"
4. Go to parse (https://www.parse.com/apps/<slug_of_your_app>/edit#keys), copy Application ID, Rest API Key, Master API
5. Click on "Settings" in wordpress
6. Set Application ID, Rest API Key, Master API and Max length of message.
7. Click on the menu item "Send Parse Push"
8. Set message for users
9. Select your audience
10. Click on send push
